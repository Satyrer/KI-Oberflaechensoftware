# n8n-Workflow-Kopien

Diese Dateien sind dokumentierte Kopien der n8n-Workflows, die zur KI-Oberflaechensoftware gehoeren.

## Dateien

- `n8n_rag_build.json`: RAG-Aufbau im lokalen KI/n8n-Stack.
- `n8n_schreib_ai_review.json`: SchreibAI-Review-Workflow.
- `n8n_schreib_ai_transcription.json`: Webhook fuer Media-Transkription aus dem Deploy-Paket.

## Quellen

Die RAG- und Review-Workflows stammen aus dem lokalen KI/n8n-Stack:

```text
F:\Virtuelle Maschinenen\Eigene AI\Projekt\eigene-ai-rag\workflows\
```

Der Transkriptions-Workflow stammt aus:

```text
deploy\schreibai-transcription\workflows\
```

Wenn produktive Workflow-Dateien geaendert werden, sollen die Kopien in diesem Ordner mit aktualisiert werden.
